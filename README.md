
## Setup & Run

1. **Clone the repo:**

```bash
git clone https://github.com/alokrajmindfire/backend-dam.git
cd backend-dam
```

2. **Install dependencies with Poetry:**

```bash
poetry install
```

3. **Activate virtual environment:**

```bash
poetry env activate
```

4. **Run the app:**

```bash
poetry run poe dev
```

5. **Open in browser:**
   [http://127.0.0.1:8000](http://127.0.0.1:8000)
